# Sports Trivia Game 🏆

A full-stack sports trivia game with 150 questions across multiple sports categories.

## Tech Stack
- **Frontend**: React
- **Backend**: Node.js + Express
- **Database**: PostgreSQL

## Features
- 15 random questions per game from a pool of 150+
- 20-second timer per question
- Score based on answer speed (faster = more points)
- Global leaderboard

## Running Locally

```bash
docker compose up --build
```

- Frontend: http://localhost:3000
- Backend API: http://localhost:4000

## Sports Categories
Football, Basketball, Tennis, Formula 1, Olympics, Cricket, American Football, Baseball, Golf, Swimming, Boxing, Athletics, Cycling, Rugby, Hockey, Volleyball
